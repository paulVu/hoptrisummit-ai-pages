🔌 Tích hợp kênh Zalo OA & Facebook Messenger

Zalo OA

1
Tạo Zalo OA App

Truy cập oa.zalo.me → Tạo ứng dụng → Lấy App ID, App Secret, OA Access Token

2
Cấu hình Webhook

Set webhook URL: https://your-server/webhooks/zalo
Subscribe events: user_send_text, follow, unfollow

3
Thêm biến môi trường
ZALO_APP_ID=your_app_id
ZALO_APP_SECRET=your_secret
ZALO_OA_ACCESS_TOKEN=your_token
4
Luồng xử lý

User gửi tin nhắn → Zalo Webhook → Adapter chuyển đổi → POST /api/chat → Gửi reply về Zalo API

Facebook Messenger

1
Tạo Meta App

Truy cập developers.facebook.com → Tạo App → Thêm Messenger product → Kết nối Page

2
Cấu hình Webhook

Callback URL: https://your-server/webhooks/messenger
Verify Token: tự đặt
Subscribe: messages, messaging_postbacks

3
Thêm biến môi trường
META_PAGE_ACCESS_TOKEN=your_token
META_APP_SECRET=your_secret
META_VERIFY_TOKEN=your_verify
4
Luồng xử lý

User gửi message → Meta Webhook → Adapter chuyển đổi → POST /api/chat → Gửi reply về Messenger Send API
